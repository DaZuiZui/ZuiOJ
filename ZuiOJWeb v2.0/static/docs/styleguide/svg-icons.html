SVG Icons

Make use of this exclusive pack of premium SVG icons included in the theme’s pack

SVG

Quick comes with a beautiful set of premium SVG icons.

Usage

There are a few ways in which you can use these icons:

  • as static images
  • injecting them directly into the DOM via the <img> tag using the <a href=”../plugins/svg-injector.html”SVG Injector</a> plugin

Location

All the icon files are located in assets/img/svg/icons/ folder.
